JKL

Introducing JKL - a lightweight boilerplate for creating a responsive Jekyll blog. Minimal styles focused on readability. You shouldn't have to do busy work to get going.

Typography is ace. Default body copy is set to 20px. Works on big desktops and small mobile phones.
